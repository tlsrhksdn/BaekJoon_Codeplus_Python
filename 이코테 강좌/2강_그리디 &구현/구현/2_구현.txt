구현 유형 예시

알고리즘은 간단하지만 코드가 지나칠 만큼 길어지는 문제
실수 연산을 다루고, 특정 소수점 자리까지 출력해야 하는 문제
문자열을 특정한 기준에 따라서 끊어 처리해야 하는 문제
적절한 라이브러리를 찾아서 사용해야 하는 문제

알고리즘 문제의 2차원 공간: 행렬을 통해 구현

시뮬레이션 및 완전 탐색: 2차원 공간에서의 방향 벡터를 통해 사용

# 동, 북, 서, 남
dx = [0, -1, 0, 1]
dy = [1, 0, -1, 0]

# 현재 위치
x, y = 2, 2

for i in range(4):
  # 다음 위치
  nx = x + dx[i]
  ny = y + dy[i]
  print(nx, ny)
